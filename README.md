# KoshiPad

KoshiPad is a compact 9-key macropad. It features a 128×32 OLED display, per-key RGB backlighting, a rotary encoder, and a custom-designed PCB and enclosure.

## Features

- 9 mechanical keys
- 128×32 OLED display
- 9× SK6812 MINI-E per-key RGB LEDs
- EC11 rotary encoder
- Seeed XIAO RP2040
- Custom PCB designed in KiCad
- Two-piece 3D printable enclosure
- Snap-fit lip-and-groove case design
- KMK Firmware

## CAD Model

The enclosure consists of two printed parts:

- Top case
- Bottom case

The PCB is mounted inside the bottom case using M3 hardware, while the top and bottom enclosure are secured using a snap-fit lip-and-groove mechanism for a clean, screw-free exterior.
